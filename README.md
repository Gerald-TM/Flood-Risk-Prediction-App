# ECOLAG: Flood Risk Predictor

An interactive geospatial dashboard that identifies flood-prone areas across Lagos LGAs.

## Features
- Normalized environmental factors
- LGA and flood risk level analysis
- Waste collection efficiency analysis
- Flood risk visualization map


## Tech Stack
- Python
- Streamlit
- Pandas
- Folium

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
